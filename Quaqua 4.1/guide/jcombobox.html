Size style

Quaqua supports regular sized and small sized combo boxes. It is also possible to specify larger font sizes for a JComboBox, but the results are not necessarily optimal.


You can change the size of a combo box by setting an appropriate font. Combo boxes are displayed with regular size for the font Lucida Grande 13 and with small size for the font Lucida Grande 11.

 

myComboBox.setFont(

   "Lucida Grande", Font.PLAIN, 11

);
